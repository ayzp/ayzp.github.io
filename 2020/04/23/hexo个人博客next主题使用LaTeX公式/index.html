前言

在使用next主题的过程中，碰到写的markdown中有LaTeX公式不显示的问题，遂查找资料解决。现已解决，将笔记记录于此。使用的next主题版本是Release v7.8.0。

第一步 更换Hexo默认渲染引擎

hexo 默认的渲染引擎是 marked，但是 marked 不支持 mathjax。所以需要更换Hexo的markdown渲染引擎为hexo-renderer-kramed引擎，后者支持mathjax公式输出。
